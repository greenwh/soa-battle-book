# Pre-Battle Checklist

::: info Quick take
The difference between a prepared player and an unprepared one is often bigger than the difference in their hero development. Run this checklist before every competitive event.
:::

## Permanent Buffs

These should always be in place. Check once, maintain ongoing — you shouldn't need to fix these before each event.

- [ ] **Hero gear** enhanced and equipped on correct heroes (priority slots at Mythic)
- [ ] **Hero skills** leveled — Expedition skills for troop combat, Exploration for Arena
- [ ] **Chief gear** equipped and enhanced (all 6 pieces for set bonus) — unlocks at Furnace 22
- [ ] **Chief charms** leveled evenly (uniformity bonus) — unlocks at Furnace 25
- [ ] **VIP status** active (10,000 gems/30 days minimum)
- [ ] **Skins equipped** — city skins (+2% Troop Attack each, stacking) and teleport skins (+2% Troop Lethality each, stacking)
- [ ] **Battle research** tree current in Academy
- [ ] **Alliance tech** contributions maintained (prioritize Battle tree when officer)
- [ ] **Daybreak Island** decorations upgraded, Essence collected twice daily
- [ ] **All troops** promoted to highest available tier (dismiss lower-tier troops)

## Timed Buffs — Activate Before Event

Pop these 5–10 minutes before the event starts. Pet skills last 2 hours; gem buffs last 2 or 12 hours depending on which you use.

### Pet Combat Skills

Activate the pets relevant to your event role:

| Pet | Skill | Effect | When to Use |
|---|---|---|---|
| **Cave Lion** | Feral Anthem | +10% All Troop Attack | Always — S-tier offense |
| **Saber-Tooth Tiger** | Apex Assault | +10% Troop Lethality | Always — S-tier offense |
| **Frost Gorilla** | Earthbound Vigor | +10% Troop Health | Defensive events, garrison duty |
| **Mammoth** | Hardened Skin | +10% Troop Defense | Defensive events, garrison duty |
| **Snow Ape** | Tumbling Power | +15,000 Squad Capacity | Rally events (more troops = more impact) |
| **Iron Rhino** | Rallying Beast | +150,000 Rally Capacity | Rally leaders only |

### Gem Buffs

Available as 2-hour (cheaper) or 12-hour (better for long events like SvS).

- [ ] Troops Attack Up (+10% for 2hr / +20% for 12hr)
- [ ] Troops Lethality Up (+10% / +20%)
- [ ] Troops Defense Up (+10% / +20%)
- [ ] Troops Health Up (+10% / +20%)
- [ ] Deployment Capacity Up (+10% / +20%)

::: tip Stockpile over purchase
Stockpile free buff items from events, daily rewards, and shops rather than buying with gems. Save gem purchases for critical events like SvS Castle Battle.
:::

### Troop Ratio

- [ ] **Set troop ratio to 50/20/30** (Infantry/Lancer/Marksman) — do NOT use the default 33/33/33. You must manually adjust the sliders every time.

## Event-Specific Reminders

| Event | Extra Steps |
|---|---|
| **Alliance Championship** | Activate ALL buffs, THEN register your march. Do NOT re-register after buffs expire. |
| **Foundry Battle** | Stock 10–15 Advanced Teleporters. Assign garrison and attack heroes to separate formations. |
| **Bear Trap / Bear Hunt** | Ready 2–3 hours of healing speedups. If alliance kills bear easily, do NOT use Titan Roc Armor Rift. |
| **SvS Castle Battle** | Use 12-hour gem buffs, timed for the Castle Battle window (typically 10:00 UTC Saturday). |
| **Crazy Joe** | Coordinate troop swap with a partner before Wave 1 for double points. Infantry-heavy reinforcements. |

## The 60-Second Version

If you're short on time and can only do the essentials:

1. Pop Cave Lion + Saber-Tooth Tiger (30 seconds)
2. Set troop ratio to 50/20/30 (10 seconds)
3. Check that your correct hero is in slot 1 (10 seconds)
4. Join the event (10 seconds)

That's better than most unprepared players and takes under a minute.
